## Stack

- **Framework:** [Jekyll](https://jekyllrb.com) (Forever Jekyll 기반 커스텀 다크 테마)
- **Hosting:** GitHub Pages
- **Theme:** 다크 테마 (`#0d1117`), 사이드바 레이아웃

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

## New Post

```bash
bundle exec jekyll post "포스트 제목"
```

`_posts/YYYY-MM-DD-title.md` 파일이 생성됩니다.
