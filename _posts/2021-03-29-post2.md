업무하다 보면 터미널 한 줄에서 명령어를 전부 표시하기 어려운 경우가 종종 있습니다.
물론 Oh My ZSH 테마를 바꾸는 것도 방법이겠지요.
저는 Hostname을 가려서 좀 더 공간을 확보해보겠습니다!


### Code

{% raw %}
```
cat << EOF > ~/.zshrc
# Hide Hostname
prompt_context() {
 if [[ "$USER" != "$DEFAULT_USER" || -n "$SSH_CLIENT" ]]; then
   prompt_segment black default "%(!.%{%F{yellow}%}.)$USER"
 fi
}

EOF
```
